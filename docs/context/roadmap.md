# Roadmap Context

## V1 - Conversao simples com usuarios

Objetivo: entregar uma versao funcional rapidamente, com backend bem estruturado.

Funcionalidades:

- cadastro de usuario;
- login;
- autenticacao JWT;
- upload de PDF;
- detectar se o PDF e vetorial ou escaneado;
- extrair linhas, curvas, textos e possiveis layers;
- criar job de conversao;
- gerar DXF sem calibracao de escala;
- disponibilizar download do resultado;
- registrar historico de conversoes por usuario;
- mostrar status da conversao;
- controlar limite de conversoes por usuario.

Fora da V1:

- PDF escaneado com OCR completo;
- conversao perfeita de blocos AutoCAD;
- DWG direto;
- CDR;
- calibracao de escala;
- usuarios premium;
- conversao generica de qualquer PDF para CAD.

## V2 - Planos, creditos e escala

Objetivo: tornar o produto mais robusto e monetizavel.

Funcionalidades:

- usuarios free;
- usuarios premium;
- limites diferentes por perfil;
- compra de conversoes extras;
- saldo ou creditos de conversao por usuario;
- calibracao de escala;
- prioridade de fila para premium;
- painel de uso do usuario.

## V3 - Recursos avancados de conversao

Objetivo: expandir recursos avancados.

Funcionalidades:

- exportacao para DWG;
- conversao em lote;
- OCR para PDFs escaneados;
- organizacao por projetos;
- recursos especificos para projetos eletricos;
- melhorias na limpeza do DXF gerado.

## V4 - Storage premium e biblioteca de projetos

Objetivo: transformar o sistema em uma plataforma de trabalho com biblioteca de projetos.

Funcionalidades:

- storage premium para salvar e manter projetos;
- biblioteca de projetos por usuario;
- armazenamento de PDFs originais;
- armazenamento de DXFs gerados;
- historico completo dentro de cada projeto;
- controle de espaco usado por usuario;
- regras de retencao diferentes para usuario free e premium;
- organizacao por cliente, obra ou disciplina.
